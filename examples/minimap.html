Minimap Plugin

The minimap plugin renders a smaller overview of the full waveform. Useful for long audio files to provide navigation context.

from wavesurf import Plugins WaveSurfer( audio=audio, sr=24000, title="With Minimap", plugins=[ Plugins.minimap( height=30, wave_color="#666688", progress_color="#6c63ff", ), ], )
