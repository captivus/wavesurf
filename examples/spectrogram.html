Spectrogram Plugin

The spectrogram plugin displays a frequency spectrum visualization below the waveform. The sweep audio shows a clear rising frequency.

from wavesurf import Plugins WaveSurfer( audio=audio, sr=24000, title="With Spectrogram", plugins=[Plugins.spectrogram(height=128, labels=True)], )
